# C端用户首页重新设计 - 2026年1月29日

## 设计概述

根据提供的设计图，重新编写了C端用户首页（`packages/user-web/src/views/Home.vue`），实现了全新的布局和样式。

## 页面结构

### 1. 顶部导航栏（两部分）

#### 第一部分（Top Nav Bar）
- **背景色**：#333333
- **内容**：
  - 注册按钮（右侧）
  - 语言切换下拉菜单
  - 个人中心下拉菜单（UserAvatar组件）
- **位置**：固定在页面最顶部（z-index: 1001）

#### 第二部分（Main Header）
- **背景色**：白色（#ffffff）
- **内容**：
  - LOGO（左侧，颜色 #00699A）
  - 导航菜单（首页、商城、设计师、社区、关于）
  - 搜索框（右侧）
  - 购物车图标
- **位置**：固定在第一部分下方（top: 40px, z-index: 1000）

### 2. 中间内容区

#### 左侧菜单（Sidebar Menu）
- **背景色**：#f9fafb
- **宽度**：2列（col-md-2）
- **内容**：
  - 风格分类（现代、北欧、中式、美式、日式、轻奢）
  - 房间类型（客厅、卧室、厨房、卫生间）
- **交互**：
  - 悬停效果：背景变为 #f3f4f6
  - 激活状态：背景 #e6f4f9，左侧蓝色边框（#00699A）

#### 右侧内容区（Content Area）
- **宽度**：10列（col-md-10）
- **内容**：

##### 轮播图（Carousel）
- 高度：400px
- 圆角：8px
- 阴影效果
- 底部渐变遮罩显示标题和按钮

##### 商品列表（Product Grid）
- **布局**：两列网格（grid-template-columns: repeat(2, 1fr)）
- **间距**：1.5rem
- **商品卡片包含**：
  - 商品图片（250px高度）
  - 悬停时显示点赞和收藏按钮
  - 价格（#00699A，1.25rem，粗体）
  - 标题和描述
  - 设计师信息（头像 + 名称）
  - 浏览量统计
  - 加入购物车和立即购买按钮

##### 优惠广告横幅（Promotional Banner）
- **位置**：每4个商品后插入一个
- **布局**：横跨两列（grid-column: 1 / -1）
- **背景**：红色渐变（#ff6b6b 到 #ee5a6f）
- **内容**：
  - 左侧：促销图片（200x150px）
  - 右侧：促销文字信息
    - 标题："Promotional discount campaign"
    - 优惠详情
    - 活动时间
    - 领取方式
    - 更多活动提示
- **高亮文字颜色**：#ffd700（金色）

### 3. 底部页脚（Footer）
- **背景色**：#333333
- **文字颜色**：#9ca3af
- **内容**：
  - 版权信息
  - ICP备案号

## 颜色系统

- **主题色**：#00699A（蓝色）
- **深色背景**：#333333（导航栏和页脚）
- **浅色背景**：#f9fafb（侧边栏）
- **边框色**：#e5e7eb
- **文字色**：
  - 主要文字：#111827
  - 次要文字：#6b7280
  - 辅助文字：#9ca3af
- **促销红色**：#ff6b6b 到 #ee5a6f（渐变）
- **高亮金色**：#ffd700

## 交互效果

### 商品卡片
- 悬停时：
  - 阴影加深
  - 向上移动2px
  - 图片放大1.05倍
  - 显示点赞和收藏按钮

### 按钮
- 加入购物车：白色背景，悬停时边框和文字变为主题色
- 立即购买：主题色背景，悬停时颜色加深

### 菜单项
- 悬停时背景变浅
- 激活时显示左侧蓝色边框

## 响应式设计

- 移动端（<768px）：
  - 商品网格改为单列
  - 促销横幅改为垂直布局
  - 促销图片宽度100%

## 技术实现

- **框架**：Vue 3 + TypeScript
- **样式**：Scoped CSS
- **布局**：Bootstrap Grid + CSS Grid
- **图标**：Bootstrap Icons
- **组件**：
  - LanguageSwitcher（语言切换）
  - UserAvatar（用户头像）

## 文件路径

`SIA-clone/packages/user-web/src/views/Home.vue`

## 特点

1. **固定导航栏**：顶部导航固定，滚动时始终可见
2. **两列商品布局**：清晰展示商品信息
3. **穿插广告**：每4个商品后插入促销横幅
4. **流畅交互**：悬停效果和过渡动画
5. **响应式设计**：适配不同屏幕尺寸
6. **统一配色**：遵循设计规范的颜色系统
