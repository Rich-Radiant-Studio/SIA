# Railway 部署指南（最简单的方案）

Railway 是最简单的部署平台，一键部署，自动识别配置。

## 快速部署

### 1. 访问 Railway
https://railway.app

### 2. 一键部署
1. 点击 "Start a New Project"
2. 选择 "Deploy from GitHub repo"
3. 选择 `Rich-Radiant-Studio/SIA` 仓库
4. Railway 会自动检测并部署

### 3. 配置环境变量（如果需要）
在项目设置中添加环境变量

### 4. 获取公网地址
部署完成后会自动生成公网地址
